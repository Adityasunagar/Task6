# Portfolio Website (Flask)

## Objective
Create a simple personal portfolio website using Flask, HTML, and CSS.

## Features
- Homepage with personal info
- Projects section
- Contact form with POST request
- Flask backend handling form data

## Folder Structure
portfolio/
├── app.py
├── static/styles.css
└── templates/
    ├── index.html
    └── contact.html

## How to Run
1. Install Flask:
   pip install flask

2. Run the server:
   python app.py

3. Open in browser:
   http://127.0.0.1:5000/

## Outcome
Basic understanding of web development using Flask with routing, templates, and forms.
